# ZirconOS Classic Wallpapers

The Classic (Windows 2000) theme uses **solid color desktop backgrounds** rather
than wallpaper images. This matches the original Windows 2000 behavior where
the default desktop was a solid teal (#008080) color fill.

## Default Desktop Colors by Scheme

| Scheme | Color | Hex | Description |
|--------|-------|-----|-------------|
| Windows Standard | Teal | `#008080` | Default Windows 2000 desktop color |
| Storm | Black | `#000000` | Dark monochrome desktop |
| Spruce | Forest Green | `#006040` | Green nature-inspired desktop |
| Lilac | Violet | `#604080` | Purple-themed desktop |
| Desert | Sand | `#C09850` | Warm sandy desktop |
| High Contrast Black | Black | `#000000` | Accessibility — black background |
| High Contrast White | White | `#FFFFFF` | Accessibility — white background |

## Notes

- No wallpaper image files are shipped with the Classic theme
- The desktop background is rendered as a solid color fill via GDI
- Users may optionally set a BMP wallpaper through Display Properties
- Wallpaper tiling modes (Tile, Center, Stretch) are supported at the
  compositor level but not used by default
