## Design Language

All icons follow the Windows 2000 Classic visual language:

- **16-Color Palette**: Standard VGA 16-color system palette
- **Flat Shading**: No gradients — solid color fills with dithering
- **1px Outlines**: Black outlines for shape definition
- **No Anti-aliasing**: Pixel-perfect edges, no smoothing
- **32x32 Grid**: All icons designed on a 32x32 pixel grid
